speculate
=========

[![GitHub release](https://img.shields.io/github/release/akerl/speculate.svg)](https://github.com/akerl/speculate/releases)

Tool for assuming roles in AWS accounts

## Usage

To assume the "admin" role in the local account:

```
speculate admin
```
